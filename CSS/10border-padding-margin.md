# 内边距、边框、外边距

元素框描述了一个元素在文档布局中所占的空间大小，每个框影响其他元素框的位置和大小

默认（正常文档流中），一个显示的文档是由多个矩形框组成的，这些矩形框分不开，从而不会相互重叠；如果使用定位、浮动，或在正常流使用负的外边距，都可能出现视觉重叠。

元素的`width`，`height`，对于这两个属性不能应用于行内非替换元素

元素与元素之间的空间：可以通过设置内边距、外边距。在没有背景的情况下，他们没有多少区别
